# TabBar 图标添加说明

## 图标要求

微信小程序 tabBar 图标需要满足以下要求：

1. **尺寸**：建议 81px × 81px（实际显示时会缩放）
2. **格式**：PNG 格式
3. **颜色**：
   - 普通状态图标：使用灰色（#7A7E83）
   - 选中状态图标：使用蓝色（#1989fa）
4. **透明度**：支持透明背景

## 需要添加的图标文件

请在 `assets/images/` 目录下添加以下 8 个图标文件：

### 时间线
- `timeline.png` - 未选中状态
- `timeline-active.png` - 选中状态

### 待办
- `todo.png` - 未选中状态
- `todo-active.png` - 选中状态

### 其他
- `other.png` - 未选中状态
- `other-active.png` - 选中状态

### 个人
- `profile.png` - 未选中状态
- `profile-active.png` - 选中状态

## 添加图标后的配置

添加图标文件后，需要在 `app.json` 中恢复图标配置：

```json
"tabBar": {
  "list": [
    {
      "pagePath": "pages/timeline/index",
      "iconPath": "assets/images/timeline.png",
      "selectedIconPath": "assets/images/timeline-active.png",
      "text": "时间线"
    },
    {
      "pagePath": "pages/todo/index",
      "iconPath": "assets/images/todo.png",
      "selectedIconPath": "assets/images/todo-active.png",
      "text": "待办"
    },
    {
      "pagePath": "pages/other/index",
      "iconPath": "assets/images/other.png",
      "selectedIconPath": "assets/images/other-active.png",
      "text": "其他"
    },
    {
      "pagePath": "pages/profile/index",
      "iconPath": "assets/images/profile.png",
      "selectedIconPath": "assets/images/profile-active.png",
      "text": "个人"
    }
  ]
}
```

## 图标资源推荐

1. **IconFont（阿里巴巴图标库）**
   - 网址：https://www.iconfont.cn/
   - 搜索关键词：时间、待办、其他、个人

2. **Flaticon**
   - 网址：https://www.flaticon.com/
   - 提供大量免费图标

3. **Icons8**
   - 网址：https://icons8.com/
   - 提供多种风格的图标

4. **自定义设计**
   - 使用 Figma、Sketch 等设计工具
   - 确保图标风格统一

## 临时方案

目前 tabBar 已配置为仅显示文字，项目可以正常运行。添加图标后按照上述说明更新 `app.json` 即可。
